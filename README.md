# QA & Security Demo Page

Це простий проект — демонстрація HTML + CSS з використанням Flexbox та Grid.

## Файли
- `index.html` — головна сторінка
- `styles.css` — стилі (CSS variables, Flexbox, Grid)
- `screenshots/` — приклади скріншотів (mockups)

## Як відкрити
Відкрий `index.html` у браузері (двічі клацни файл або правою кнопкою → Open with...).

---

Цей репозиторій підходить як вправа для QA: перевірити адаптивність, поведінку кнопок, порядок елементів, семантику та доступність.
